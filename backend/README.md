Before running the project, you need to follow these steps:

### 1. Create a virtual environment

```bash

```

### 2. Activate the virtual environment

**On Windows:**
```bash

```

**On macOS/Linux:**
```bash
source env/bin/activate
```

### 3. Install dependencies

```bash

```
